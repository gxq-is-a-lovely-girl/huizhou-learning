# 运算符

## 1.逻辑&（与/且）

```cpp
//1010101010101
//0101010101010
//进行与运算
//0000000000000
//即
//1&0=false
//1&1=true
//0&0=flase
```

## 2.逻辑|（或）

```cpp
//1010101010101
//0101010101010
//进行或运算
//1111111111111
//即
//1&0=true
//1&1=true
//0&0=flase
```

## 3.逻辑!（非）

```cpp
//1010101010
//进行非运算
//0101010101
//即
//!1=0;
//!0=1;
//按位取反
```
