# 字符串数组

## 1.字符数组的复制

```cpp
char a[100]={"123"};
char b[100]={"abc"};
strcpy(a,b);//把b字符数组给a字符数组
```

## 2.字符数组的拼接

```cpp
char a[100]={"123"};
char b[100]={"abc"};
strcat(a,b);//b字符数组接到a字符数组，/0开始拼
```

> ## 注：每个函数都会有一个double的返回值

## 3.字典序比较

```cpp
char a[100]={"123"};
char b[100]={"abc"};
strcmp(a,b);//从第一个字符开始比较，直到比较不相同
//返回值：a>b返回值>0;a<b返回值<0;相同返回值=0
```
